# 🫀 Wykrywanie wad serca z wykorzystaniem regresji logistycznej

Projekt przedstawia implementację regresji logistycznej do wykrywania chorób serca na podstawie danych medycznych. Model został zaimplementowany samodzielnie, bez użycia gotowych bibliotek do uczenia maszynowego.

## Cel projektu

* przewidywanie występowania chorób serca,
* analiza i przygotowanie danych,
* ocena skuteczności modelu za pomocą wybranych metryk,
* porównanie wyników z innymi algorytmami klasyfikacji.

## Wykorzystane dane

Do trenowania i testowania modelu wykorzystano zbiór **Heart Disease Dataset** pochodzący z repozytorium [UCI](https://archive.ics.uci.edu/dataset/45/heart+disease).

## Metryki oceny

* Accuracy
* Precision
* Recall
* ROC-AUC
